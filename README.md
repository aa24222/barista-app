# ☕ Barista App

A web application for exploring and managing coffee and tea recipes, built with React and TypeScript.

## Features

- Browse a catalogue of coffee and tea recipes
- View detailed recipe instructions
- Responsive design for mobile and desktop

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **CSS** - Custom styling

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aa24222/barista-app.git
cd barista-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
